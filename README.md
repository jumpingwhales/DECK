# Install guide:
To download, copy the SSH address of this repo in the green Code tab. Then, open the terminal and type ```git clone SSH_ADDRESS```, replacing the SSH_ADDRESS with the copied SSH address from Github. Still in the terminal, change directories (cd) until you get to the cloned repo, and type ```python app.py```. This will open up a live server that runs on your computer. In a browser, type ```127.0.0.1:5000``` and you have reached our website LogDECK!
