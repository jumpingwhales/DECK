# Description:

# Install guide:
To clone the repo, open the terminal and enter:

```git clone git@github.com:jumpingwhales/DECK.git```

To install Flask:

```python -m pip install Flask```

To open a virtual environment:

```python3 -m venv venv```

Activate the venv:

For Windows: ```venv\Scripts\activate```

For Linux: ```. venv/bin/activate```

Install dependencies:

```pip install -r requirements.txt```

# Launch codes:
enter DECK:

```cd DECK```

in the terminal (once you are sure you are in the DECK repo):

```python app/__init__.py```

then, in a browser, enter:

```http://127.0.0.1:5000```
